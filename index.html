Introdução ao HTML

TAGS - Estruturais
<html>...</html> A tag HTML, abreviação de HyperText Markup Language, é usada para estruturar o conteúdo de uma página da web.
<head>...</head> é uma seção obrigatória em um documento HTML e é usada para conter metadados sobre o documento
<title>...</title> Define o título da página, que é exibido na aba do navegador.
<body>...</body> é uma das principais tags usadas no HTML para definir o conteúdo principal de uma página da web.
<main>...</main> é usada para representar o conteúdo principal de um documento HTML.
<body>...</body> é uma das principais tags usadas no HTML para definir o conteúdo principal de uma página da web.
